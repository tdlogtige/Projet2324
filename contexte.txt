
§u:eqt de la chaleur
§a:Pour écrire l'équation de la chaleur en code LaTeX, vous pouvez utiliser l'environnement "equation" ou "align" combiné avec les commandes mathématiques appropriées.

Voici un exemple d'équation de la chaleur écrit en code LaTeX :

\begin{equation}
\frac{\partial u}{\partial t} = \alpha \frac{\partial^2 u}{\partial x^2}
\end{equation}

Dans cet exemple, "u" représente la variable de température, "t" représente le temps, "x" représente la position spatiale, et "α" représente le coefficient de diffusion thermique.

N'oubliez pas d'inclure le package "amsmath" dans votre document LaTeX pour pouvoir utiliser les environnements et commandes mathématiques nécessaires.
§u:eqt de shrodinger
§a:Bien sûr, voici l'équation de Schrödinger en code LaTeX :

\[
i\hbar\frac{\partial}{\partial t}\Psi(\mathbf{r}, t) = \left(-\frac{\hbar^2}{2m}\nabla^2 + V(\mathbf{r}, t)\right)\Psi(\mathbf{r}, t)
\]
